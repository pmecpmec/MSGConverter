# START HIER – Projectdefinitie

De projectdefinitie is opgebouwd volgens de Windesheim-template (Opzet Projectdefinitie, WFSDAD.CM1.01/CM2.01). De structuur volgt de acht hoofdstukken uit die template.

## Hoofddocument

**`projectdefinitie.md`** – Alle inhoud in één bestand (Inleiding + hoofdstukken 1–8). Bewerk dit bestand voor wijzigingen.

## Word-document genereren

Voer uit vanuit de repo-root:

```
python scripts/export_projectdefinitie_to_word.py
```

Output: `Projectdefinitie-MSG3-Maximo-Converter.docx` in deze map.

## Taalgebruik

Helder en zakelijk. Correcte taal (max 10 fouten per pagina). Onbekende begrippen worden eerst uitgelegd. Derde persoonsvorm waar mogelijk. Geen emoji's, geen em-dashes.
